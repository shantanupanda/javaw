# javaw-cookbook

Java wrapper cookbook to install openjdk on ubuntu and other flavours of linux

## Supported Platforms

Ubuntu
Rhel etc.

## Usage

### javaw::default

Include `javaw` in your node's `run_list`:

```json
{
  "run_list": [
    "recipe[javaw::default]"
  ]
}
```
